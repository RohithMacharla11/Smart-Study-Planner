# 📚 Smart Study Planner with Deadline Alerts

## 📌 Project Overview
This project is a GUI-based study planner developed using Python.  
It helps students organize their tasks, track deadlines, and manage their study schedule efficiently.  

The application also includes a **deadline alert system** that notifies users about overdue tasks.

---

## 🎯 Objectives
- Manage study tasks in an organized way  
- Track deadlines effectively  
- Identify overdue tasks instantly  
- Provide a simple and user-friendly interface  

---

## 🛠️ Tools & Technologies Used
- Python  
- Tkinter (GUI library)  
- Datetime module  

---

## ✨ Features
- Add tasks with subject, deadline, and priority  
- View all pending tasks  
- Mark tasks as completed  
- Delete tasks  
- Automatically highlights overdue tasks (⚠)  
- Separate display for pending and completed tasks  
- **Deadline alert popup for overdue tasks**

## ▶️ How to Run the Project

1. Make sure Python is installed  
2. Open terminal in the project folder  
3. Run the following command:

```bash
python study_planner.py
```
4. The GUI window will open
---

## 🚨 Deadline Alert Feature
- The application automatically checks for overdue tasks when it starts  
- If any task is overdue, a popup alert is displayed  
- Helps users take immediate action on missed deadlines
